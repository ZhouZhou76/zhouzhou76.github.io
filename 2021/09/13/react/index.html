React 初识

概念

React是Facebook开发出的一款声明式，高效且灵活的用于构建用户界面的JavaScript库。使用React可以将一些简短、独立的代码片段组合成复杂的UI界面，这些代码片段被称为”组件”。

React三大体系：
1.React.js 用于Web开发和组件的编写。
2.ReactNative 用于移动端开发
3.ReactVR 用于虚拟现实技术的开发

特点

  1. React拥有较高的性能，代码逻辑非常简单。
  2. 声明式设计（采用声明范式）。
  3. 高效，通过对DOM的模拟，最大限度地减少与DOM的交互。
  4. 灵活，React可以与已知的库或框架很好地配合。
  5. JSX语法，JSX是JavaScript和xml的结合。
  6. 组件，通过React构建组件，使得代码更加容易得到复用，能够很好的应用在大项目的开发中。
  7. 单向响应的数据流，React实现了单向相应的数据流，从而减少了重复代码。

缺点

……

引入

  1. react.js (核心文件)
  2. react-dom.js (渲染页面中的DOM，依赖于react核心文件)
  3. babel.js (将ES6转换成ES5，JSX语法转换成JavaScript)

下载

npm i react --s
npm i react-dom --s
npm i babel-standalone --s

安装

#定义项目所需要的各种模块及配置信息
    cnpm init -y 
#安装react
    cnpm i react react-dom babel-standalone -S
#在项目中引入react
    <script src="node_modules/react/umd/react.development.js"></script>
    <script src="node_modules/react-dom/umd/react-dom.development.js"></script>
    <script src="node_modules/babel-standalone/babel.min.js"></script>

多行标签

只能有一个根标签。
标签必须闭合。
标签首字母：
    1.若小写字母开头，则将改标签转为html中同名元素，若html中没有该对应的同名元素，则报错。
    2.若大写字母开头，react就去渲染对应的组件，若组件没有定义，则报错。

一.表达式：一个表达式会产生一个值，可以放在任何一个需要值的地方

        4. arr.map()
        5. function test () {}
二.语句(代码)：
    如：

组件实例的三大核心属性

state

1.state是组件对象最重要的属性，值是对象（可以包含多个key-value的组合）。
2.组件被称为“状态机”，通过更新组件的state来更新对应的页面显示（重新渲染组件）。
3.组件中render方法中的this为组件实例对象。
4.组件自定义的方法中this为undefined，可通过强制绑定this（函数对象的bind()）或箭头函数。
5.状态数据，不能直接修改或更新。
