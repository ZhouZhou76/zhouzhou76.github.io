let 与 const

var     关键字声明变量,将变量或者形参所在的函数的大括号作为作用域处理,并且遵循垃圾回收进制。
let     关键字声明的变量,只要遇到大括号就形成作用域,所在的作用域叫做块级作用域。
const   声明变量,变量值只能在声明的时候确定,后面没有办法修改。不能改的变量叫做常量。
