JavaScript 面向对象

面向对象：继承(分类更加细分的构造函数),封装(封装构造函数),多态。

面向对象是一个编程思想,支撑面向对象编程思想的语法是类(ECMA6)和对象,构造函数充当类的角色。
构造函数和对象实现面向对象程序的时候,体现出继承,封装,多态的特点。
