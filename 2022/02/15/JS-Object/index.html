JS 对象学习总结

发表于2022-02-15|更新于2022-02-16
|字数总计:2.3k|阅读时长:7分钟|阅读量:

JS 对象的分类

graph LR A[JS中的对象]==>C1(宿主对象) A==>C2(JS里的对象) C1==>D1(DOM对象) C1==>D2(BOM对象) C2==>ro1(内置对象) C2==>ro2(自定义对象)
在js里，一切皆为或者皆可以被用作对象。
宿主对象：浏览器提供的对象，对象不是JavaScript语言本身的，但是可以在JavaScript中使用的对象，如DOM，BOM。
    DOM对象：HTML标签寄放在javascript中，用JavaScript可以操作HTML标签。
    BOM对象：浏览器窗口及其浏览器的组成部分寄放在JavaScript中，可以用JavaScript操作浏览器窗口和它的组成部分。
JS中的对象：
    内置对象：不必明确实例化的对象，已被实例化了。如：Math。函数中的arguments和this；事件处理函数中的event对象等等，直接使用，不用new。
    自定义对象：程序员自己创建的对象，如：Date，Array，Set，Map，XMLHttpRequest，RegExp，Promise等等要使用，必须new。

内置对象

内置对象图
JavaScrip的所有内置对象都是继承于Object对象，继承了所有的属性和方法。
对象只是一种特殊类型的数据，并拥有属性和方法，属性是与对象相关的值，方法是能够在对象上执行的动作。
JS内置对象也被称为内部类。可以说，object类是所有类的基础类。JavaScript的所有类都是在object类上面派生出来的。

对象名称 说明
Arguments 函数参数集合
Array 数组对象，使用单个变量来存储一系列的值。
Boolean 布尔对象，用于将非布尔型的值转换为布尔值(true或false)
Date 用来处理日期和时间，通过new关键字来定义Date对象。
Function 用来表示开发者定义的任何参数。
Math 具有数学常数和函数的属性和方法，用于执行常见的算数任务。
Number 原始数值的包装对象，可以自动在原始数值和对象之间进行转换。
RegExp 用于完成有关正则表达式的操作和功能。
String 字符串对象，用于处理文本。
Error 错误对象，用于错误处理。

基类Object 常用属性和方法

属性和方法 说明
prototype属性 对该对象的原型的引用。原型是一个对象，其他对象可以通过它实现属性的继承。
constructor()方法 构造函数，构造函数是类的一个特殊函数，当创建类的对象实例时，系统会自动调用构造函数，通过构造函数对类进行初始化操作。
valueOf() 返回对象的原始值
toString()方法 返回对象的字符串表示
toLocaleString()方法 返回对象地方化的字符串表示
isPrototypeOf(object)方法 检查对象是否是指定对象的原型
hasOwnProperty(proName)方法 检查对象是否有局部定义的，具有特定名字的属性
PropertyIsEnumerable(proName)方法 返回布尔值，指出所指定的属性（proName）是否为一个对象的一部分以及该属性是否是可列举的。如果proName存在于object中并且可以使用for…in循环穷举出，那么则返回True，否则返回False
delete操作 删除对象的自由属性，不能删除继承属性
assign( )方法 用于将所有可枚举属性的值从一个或多个源对象复制到目标对象，返回目标对象。
entries( )方法 返回一个给定对象自身可枚举属性的键值对数组，其排列与使用for…in 循环遍历该对象时返回的顺序一致。

继承于Object对象 常用属性和方法

Arguments

属性和方法 说明
Arguments 一个函数的参数和其他属性。
arguments[ ] 函数参数的数组。
Arguments.callee 当前正在运行的函数。
Arguments.length 传递给函数的参数个数。

Array

属性和方法 说明
Array 对数组的内部支持。
Array.concat( ) 拷贝原数组，生成新数组。。
Array.join( ) 将数组元素连接起来以构建一个字符串。
Array.length 数组的大小。
Array.pop( ) 从数组末尾取下一个元素。
Array.push( ) 给数组的末尾添加元素。
Array.reverse( ) 反转数组。
Array.shift( ) 从数组的头部取下一个元素。
Array.unshift( ) 从数组的头部新增元素。
Array.slice( ) 可以基于当前数组获取指定区域元素，提取出元素生成新数组。
Array.sort( ) 数组排序。默认从小到大排序,按照字符串排序。
Array.splice( ) 插入，删除或替换数组的元素。
Array.toString( ) 将数组转换成一个字符串。
Array.toLocaleString( ) 把数组转换成局部字符串。
Array.indexOf( ) 在数组中查找第一次出现的元素，返回查找到的元素下标。
Array.forEach( ) 循环数组。
Array.map( ) 映射数组。
Array.filter( ) 过滤数组元素。
Array.some( ) 在数组中查找是否有符合条件的元素,有返回true,没有返回false。
Array.every( ) 在数组中查找每一个元素是否有符合条件,符合返回true,不符合返回false。
Array.reduce( ) 归并数组。

Boolean

属性和方法 说明
Boolean 对布尔值的支持。
Boolean.toString( ) 将布尔值转换成字符串。
Boolean.valueOf( ) 布尔对象的布尔值。

Date

属性和方法 说明
Date 操作日期和时间的对象。
Date.getDate() 返回一个月中的某一天。
Date.getDay() 返回一周中的某一天。
Date.getFullYear() 返回Date对象的年份字段。
Date.getHours() 返回Date对象的小时字段。

Function

属性和方法 说明
Function JavaScript的函数。
Function.apply( ) 将函数作为一个对象的方法调用。
Function.arguments[] 传递给函数的参数。
Function.call( ) 将函数作为对象的方法调用。
Function.caller 调用当前函数的函数。
Function.length 已声明的参数的个数。
Function.prototype 对象类的原型。
Function.toString( ) 把函数转换成字符串。

Math

属性和方法 说明
Math 算术函数和常量。
Math.abs( ) 计算绝对值。
Math.acos( ) 计算反余弦值。
Math.asin( ) 计算反正弦值。
Math.atan( ) 计算反正切值。
Math.atan2( ) 计算从x轴到一个点之间的角度。
Math.ceil( ) 对一个数上舍入。
Math.cos( ) 计算余弦值。
Math.E( ) 算术常量e。
Math.exp( ) 计算e^x。
Math.floor( ) 对一个数下舍入。
Math.LN10( ) 表示 10 的自然对数，约为 2.302。
Math.PI 180弧度。
Math.max() 返回最大的参数。
Math.min() 返回最小的参数。
Math.log() 计算一个数的自然对数。
Math.pow() 计算幂。
Math.random() 返回0-1之间的随机数。
Math.round() 四舍五入(成整数,只看小数后一位)。
Math.sin() 计算正弦值。
Math.sqrt() 计算平方根。

Number

属性和方法 说明
Number 对数字的支持。
Number.MAX_VALUE 最大数值。
Number.MIN_VALUE 最小数值。
Number.NaN 特殊的非数字值。
Number.NEGATIVE_INFINITY 负无穷大。
Number.POSITIVE_INFINITY 正无穷大。
Number.toExponential( ) 用指数计数法格式化数字。
Number.toFixed( ) 采用定点计数法格式化数字。
Number.toLocaleString( ) 把数字转换成本地格式的字符串。
Number.toPrecision( ) 格式化数字的有效位。
Number.toString( ) 将一个数字转换成字符串。
Number.valueOf( ) 返回原始数值。

RegExp

属性和方法 说明
RegExp 用于模式匹配的正则表达式。
RegExp.exec( ) 通用的匹配模式。
RegExp.test( ) 检测一个字符串是否匹配某个模式。
RegExp.toString( ) 把正则表达式转换为字符串。
RegExp.global 正则表达式是否全局匹配。
RegExp.ignoreCase 正则表达式是否区分大小写。
RegExp.lastIndex 下次匹配的起始位置。
RegExp.source 正则表达式的文本。

String

属性和方法 说明
String 对字符串的支持。
String.charAt( ) 返回字符串中的第n个字符。
String.charCodeAt( ) 返回字符串中的第n个字符的代码。
String.concat( ) 连接字符串。
String.fromCharCode( ) 从字符编码创建一个字符串。
String.indexOf( ) 检索字符串。
String.lastIndexOf( ) 从后向前检索一个字符串。
String.length 字符串的长度。
String.localeCompare( ) 用本地特定的顺序来比较两个字符串。
String.match( ) 找到一个或多个正则表达式的匹配。
String.replace( ) 替换一个与正则表达式匹配的子串。
String.search( ) 检索与正则表达式相匹配的子串。
String.slice( ) 抽取一个子串。
String.split( ) 将字符串分割成字符串数组。
String.substring( ) 返回字符串的一个子串。
String.toLocaleLowerCase( ) 把字符串转换小写。
String.toLocaleUpperCase( ) 把字符串转换大写。
String.toString( ) 返回字符串。
String.valueOf( ) 返回字符串。

Error

属性和方法 说明
Error 普通异常。
Error.message 可以读取的错误信息。
Error.name 错误的类型。
Error.toString( ) 把错误对象转换为字符串。
EvalError 在不正确使用eval()时抛出。
SyntaxError 抛出该错误用来通知语法错误。
RangeError 在数字超出合法范围时抛出。
ReferenceError 在读取不存在的变量时抛出。
TypeError 当一个值的类型错误时，抛出该异常。
URIError 由URI的编码和解码方法抛出。
